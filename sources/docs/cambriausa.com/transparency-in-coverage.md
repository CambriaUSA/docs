# Transparency in Coverage Files

---

This [linkopens in a new tab](https://www.bluecrossmn.com/transparency-coverage-machine-readable-files) leads to the machine-readable files that are made available in response to the federal Transparency in Coverage Rule. The machine-readable files are formatted to allow researchers, regulators and application developers to more easily access and analyze data.
